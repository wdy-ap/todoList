## 功能
* 西瓜图标和todo输入框切换
* 添加、修改、删除todo
* 页面和服务器同步更新，刷新页面不会丢失数据

### 功能实现
#### 核心技术
> 运用node.js和express框架搭建
#### 输入框切换实现
> 通过事件委托，点击西瓜图片和添加按钮时，分别为这两个元素切换class
#### 添加、删除、修改todo
> 为添加按钮、完成按钮、todo文本绑定事件，事件执行后拿到数据
#### 页面和服务器同步更新
> 将从页面抓取的数据作为请求通过ajax发送，服务器在回调中将数据处理，更新todoList，最后将html的页面更新
